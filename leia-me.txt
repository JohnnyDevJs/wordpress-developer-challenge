Instruções do tema:

1. Instalar as dependências de desenvolvimento dentro da pasta do tema: npm install / npm i;
2. No arquivo webpack.config.js na opção: BrowserSyncPlugin, trocar o proxy para a porta utilizada no seu localhost;
3. Rodar npm run build para abrir o host http://localhost:3000 onde está rodando o tema;
4. Estou enviando o arquivo do banco de dados em sql (bx_desafio.sql) e a pasta uploads caso queiram utilizar;
